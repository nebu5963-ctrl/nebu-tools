# nebu-tools

公開用Webツール集。**このリポジトリはPublicリポジトリ**であることを常に意識すること。

## 構成

```
nebu-tools/
  ├── README.md
  └── <tool-name>/
      └── index.html   ← 単一HTMLファイルで完結
```

ツールを追加するときは `<tool-name>/index.html` の形でフォルダを作る。

## ルール（絶対）

- **個人情報・秘密情報を一切含めない**：氏名、メールアドレス、Firebase/API設定（apiKey, databaseURL等）、個人用キャラクター画像、その他特定の個人に紐づく情報。
- **外部サービスへのデータ送信なし**：データ保存は `localStorage` のみ。サーバー同期が必要な機能は載せない。
- **`artifacts`（個人用・非公開リポジトリ）にある近い機能のツールを元にする場合**：
  - Firebase関連のコード（SDK読み込み・初期化・認証）を全部除去し、`localStorage` ベースの読み書きに置き換える。
  - 個人用のアニメーション・画像・キャラクター要素を除去する。
  - 個人を特定できるコメント・メモを削除する。
  - 機能面はFirebase依存以外はできるだけ元のまま保つ（劣化版ではなく「個人情報を抜いた版」という位置づけ）。
- コミット前に、追加・変更したファイルに上記が含まれていないか確認する（grepで `firebase|apiKey|databaseURL|個人名` 等をチェック）。

## メンテナンス

- 各ツールは独立して動作する。他ツールへの依存を作らない。
- 修正完了後、すぐにコミットしない。ユーザーの確認を待ってからコミット・プッシュする（`artifacts` と同方針）。
